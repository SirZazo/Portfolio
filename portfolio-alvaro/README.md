# Portfolio Álvaro

Esqueleto base de portfolio en React + Vite.

## Ejecutar en local

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Personalizar

- Sustituye email, GitHub y LinkedIn en `src/components/Contact.jsx`
- Cambia los textos en los componentes dentro de `src/components`
- Ajusta colores y espaciados en `src/styles.css`
